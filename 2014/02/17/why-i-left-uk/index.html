我为什么离开英国

X's laboratory
2014-02-17
Denmark, Moving, Travel, United Kingdom

我在英国完成了硕士学位，接着，我决定做PhD，换一个地方。之中，不论是从中国的亲戚朋友，英国的旧相识，以及目前的新同学那里，我都得到了一个相同的问题：为什么不在英国读博士？

如果能用三言两语就讲清楚，我也不至于专门写下这篇文章。我试图把我目前所知道的事实以及我自己的经历描述出来。

中国籍PhD在英国的窘境

我曾经简短讲述过为什么离开英国的故事：

(1/4)英国博士生要交学费，EU/UK籍和其他国籍数额有别，后者比前者高四五倍甚至接近十倍是有的。就我当时的学校而言，带实验的理工科EU/UK学费约1600英镑每年，非EU/UK则是13000多英镑。如果阅读过一些英国的PhD招生信息，会发现许多都强调只收UK/EU国籍的学生。 — X (@qixiaoe) January 21, 2014

(2/4)原因可能(但不限于)有两种：1提供奖学金的机构明确规定只能录取EU/UK国籍学生，2老板拿到的奖学金是EU/UK rate，只能覆盖EU/UK学生的学费+生活费，负担不起高昂的外籍学生学费。如果非EU/UK籍能拿到全额，我得说是实力+运气，但有时实力过硬也未必能拿到。 — X (@qixiaoe) January 21, 2014

(3/4)在我当时小圈子的中国学生里，不少拿着EU/UK rate奖学金的同学，在全部奖学金外加房补(每年3500英镑左右)覆盖掉国际生学费之后，每月只剩400英镑左右，而英国签证要求每月最低生活费是800英镑。少数几个(5个左右)是全自费(生活费+学费全部自掏腰包)读PhD的。 — X (@qixiaoe) January 21, 2014

(4/4)那些拿着UK/EU rate的同学，他们父母会借一部分钱帮助他们完成签证，但之后多数也不好意思问父母再要钱，只能晚上去找零工做，即便家里的手头比较宽裕。问过他们其中几个是怎么想的，“有个读PhD的机会，不容易，即使奖学金少得可怜，我也会坚持下去”。我只能说我很佩服他们。 — X (@qixiaoe) January 21, 2014

事实上，在从英国得到奖学金的PhD学生里，并不是所有中国籍PhD学生都拿的UK/EU rate，拿full rate的也不是少数；他们可以从企业、欧盟或者独立基金会得到资助。甚至，出于鼓励人才流动的关系，一些资金来源(比如ERC)有个别还不能给英国学生。

临近我硕士毕业的关头，我的导师与他人合作，拿到了一笔资金，赞助的正好是我研究的课题。导师便顺水推舟地问我要不要接了这笔钱，“鉴于你非常熟悉课题，如果你要接这个offer，我们完全可以说服赞助者，让一个非UK/EU的学生拿下……只是，钱还是UK/EU rate的。”

于我而言，我的底线有以下几条：

  1. 25岁之前实现经济独立，不依靠父母提供资金过活。
  2. 用更多的时间专注研究，而不是白天研究课题晚上到餐馆刷盘子接私活。
  3. 能学习更多知识和技能，在可能的情况下多走多看。

基于自己的底线：

  1. (抵消国际生学费之后的)400英镑每个月的奖学金根本不足以使人实现经济独立。
  2. 除非找父母赞助，否则刷盘子接私活在所难免。
  3. 我有可能找到一个合适的位置，提供足够的资金，满足我的个人需求。

即便我非常幸运地拿到全额（税后1100英镑每月，外加每年3000英镑的房补），我仍然不会选择留下。下面就来讨论原因。

英国边境制度及边检经历

2012年我认识了cc，之后便开始交往。cc当时在荷兰读书，而我在英国。比较幸运的事情是，我所居住的小地方到阿姆斯特丹每天都有直飞航班。

英国不属于申根区，作为第三世界国家的人，如果要去到荷兰（申根区），我需要办理签证，能办理到的最长签证是半年内允许滞留90天。

签证的花费，精力的消耗，拒签的风险，繁复的材料……虽然我个人办的申根签证也不少（半本护照用掉了），也从未有拒签记录，然而我依旧为上述的种种琐事头疼。

除此之外，我出入英国的10次旅行中，还有过两次非常不愉快的机场边检经历（所以对我个人而言，算下来比例是20%）。我被带到小黑屋里长时间地盘问，时长超过半小时。盘问之详细让人感觉像是扒皮扒得赤裸裸：把我的行李箱翻遍，找出所有的旅行和酒店订单，一个个去查信用卡记录和详细行程，甚至做某段行程的理由也要详细说明……同时被检查的还有其他英国人，但除了翻翻箱包，也不像我这样把整个旅行细节都扒个遍。我知道这是警察例行公事，并且我自己身正不怕影子歪，坏事不做不会心虚，但这让我的感觉相当糟糕：就因为我的一张黄皮肤，一头黑头发，一本第三世界国家护照，我就一而再、再而三成为被怀疑的对象？

cc作为游客，在英国入境也并不开心：4次到英国，两次从爱尔兰首都——都柏林入境（凭有效英国旅行签证可从爱尔兰入境）几乎不被提问就轻松放行；然而在另外两座英国城市入境时，其中一个被问到详细的行程，另一个则把护照和所有旅馆订单全部复印个遍。

在我被两次搜查、cc被两次严格入境检查之后，我只觉得，即使这个国家里有许多可爱的人对我很好，但我真的不受这个国家欢迎，至少，制度上。

Summary

基于上述两点，我决定离开英国，换到另一个地方去生活。我无法想像，我仍要继续把那般生活——400英镑每个月的收入，被局限于单一课题的枯燥，繁琐的签证，严苛的边检等等——继续下去，3年。

后记

如今，如果我要乘飞机去见cc，只需要买机票，签证手续之类都不用办，旅行证件可以一直放在包里，极少会被检查，所需要做的就是预先在手机里存好登机牌，安检的时候扫描一下即可；比起这个国家的平均收入水平，我的工资并不高，但足够让我过得不错；学校的条件也很赞，不限量免费彩色激光打印、不限量免费茶和咖啡、价格低廉的食堂、多样的交流机会……这些在我之前的生活中从来都没有过。当然啦，好玩的老板和同事也是非常重要的。

不多感言，我不能预知接下来会是怎样，生活还是要继续过下去才是。
